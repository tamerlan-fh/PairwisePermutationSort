# PairwisePermutationSort
Сортировка попарными перестановками

Есть массив из 6 элементов, состоящий из уникальных цифр от 1 до 6 расположенных случайным образом. 
Требуется отсортировать исходный массив, получив в результате последовательность 1,2,3,4,5,6. 
В процессе сортировки должны соблюдаться условия:
- переставлять элементы нужно парами, не меняя порядок внутри самой пары. 
- пары, участвующие в перестановке, не должны пересекаться между собой.

Примеры перестановок
1[63][25]4 -> 125634
12[56][34] -> 123456

5[41][63]2 -> 563412
[56]34[12] -> 123456

Требование к программе: 
на входе несортированный массив длиной 6, состоящий из чисел 1-6 
на выходе вывести построчно что с чем переставлялось
В случае, если массив не удалось отсортировать, то сообщение что массив не сортируется.
